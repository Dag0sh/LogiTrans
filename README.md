# LogiTrans
 приложение swift+postgresql
 
 для запуска наобходимо подключить устройства к одной сети, поставить соответсвующий ip в файле networkManager
 
 Надо дописать в конец файла pg_hba.conf разрешение 
 
 Убрать * в файле postgresql.conf в строке  listen_addresses = '*' (можно установить один адресс)
